Welcome to SMPDF's documentation!
=================================

The SMPDF program provides tools to generate Specialized Minimal PDFs for
optimized computations.

It also provides several tools for computation, analysis and comparison of PDF
predictions, including the neccesary plots.
In particular, it provides a convenient interface for APPlgrid convolutions in
Python.
